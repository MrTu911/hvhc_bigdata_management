=====================================================
CHỨC NĂNG AI (5/5)
=====================================================

✅ Sentiment Analysis
   - Single text: POST /api/ai/sentiment {text}
   - Batch: POST /api/ai/sentiment {texts:[]}
   - Output: sentiment, confidence, keywords

✅ Training Insights
   - API: POST /api/ai/insights {data}
   - Tạo 3-5 nhận xét + đề xuất

✅ Student Recommendations
   - API: POST /api/ai/recommendations {studentData}
   - Gợi ý cải thiện cá nhân

✅ Research Summary
   - API: POST /api/ai/summary {text, maxLength}
   - Tóm tắt nghiên cứu

✅ Health Check
   - API: GET /api/ai/sentiment
   - Kiểm tra trạng thái AI service

QUICK START (30 PHÚT)
=====================================================

1. Chạy setup script:
   $ cd /home/ubuntu/hvhc_bigdata_management
   $ bash scripts/setup-ai.sh

2. Chọn Abacus AI (option 1)

3. Lấy API key:
   https://abacus.ai/app/route-llm-apis

4. Nhập API key khi được hỏi

5. Build và chạy:
   $ cd nextjs_space
   $ yarn build
   $ yarn dev

6. Test:
   $ bash ../scripts/test-ai-all.sh

7. Sử dụng trên dashboard:
   - Login: giangvien@hvhc.edu.vn / password123
   - Vào: Dashboard → Instructor → Feedback Analysis
   - Click "Tạo AI Suggestions"
