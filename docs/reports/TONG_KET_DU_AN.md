## 🏗️ KIẾN TRÚC HỆ THỐNG

### Tech Stack
**Frontend:**
- Next.js 14.2.28 (React 18.2.0)
- TypeScript 5.2.2
- Tailwind CSS 3.3.3
- Shadcn UI Components
- Recharts for data visualization

**Backend:**
- Next.js API Routes
- PostgreSQL 16 (Database)
- Prisma ORM
- NextAuth.js (Authentication)

**ML/AI:**
- MLflow (Experiment tracking)
- Python ML Engine
- Model versioning system

**Infrastructure:**
- Docker & Docker Compose
- Redis (Caching)
- AWS S3 (File storage - ready)
- Nginx (Reverse proxy - ready)

### Architecture Layers
```
┌─────────────────────────────────────────┐
│         Presentation Layer              │
│   (Next.js Frontend + UI Components)    │
└─────────────────────────────────────────┘
                   ↓
┌─────────────────────────────────────────┐
│          API Gateway Layer              │
│      (Next.js API Routes + Auth)        │
└─────────────────────────────────────────┘
                   ↓
┌─────────────────────────────────────────┐
│        Business Logic Layer             │
│  (Services, Utilities, ML Workflows)    │
└─────────────────────────────────────────┘
                   ↓
┌─────────────────────────────────────────┐
│          Data Layer                     │
│  (PostgreSQL + Redis + S3 + MLflow)     │
└─────────────────────────────────────────┘
```
